04 · Shopify metaobjects

Editor-managed transparency.

The PDP's ritual, materials, and comparison surfaces are not hard-coded. They're driven by Shopify metaobjects so the content team can update without a deploy.

NamespaceKeyTypeShape
uy ritual_steps JSON [{ index, title, body }]
uy materials JSON [{ label, body, certification? }]
uy comparison JSON [{ feature, us, typical }]
uy review_summary JSON { rating, count }

05 · Cart flow

Storefront API only. Shopify owns money.

// On first add:
useCart().add(variantId)
   ↓ POST /api/cart  { action:'add', cartId:null, variantId }
      ↓ shopifyFetch  cartCreate → cartLinesAdd
         ↓ returns { cart }
   ↓ persists cart.id to localStorage as 'uy_cart_id'
   ↓ opens <CartDrawer>

// On checkout:
<a href={cart.checkoutUrl}>Checkout · Shopify</a>
   ↓ Shopify-hosted checkout — fulfillment routes to Printify or CJ
     based on the variant's connected app.
